OnePageRESTApplication
======================

Af en toe is het handig om een eenvoudige applicatie te hebben die via REST met een server kan communiceren.
Nu zijn er verschillende oplossingen om dat te realiseren. In dit blog beschrijf ik de eerste aanzet tot een mogelijke oplossing omdat
m.b.v. verschillende moderne web technologieÃ«n te realiseren.

De applicatie bestaat uit een eenvoudige webserver. De webserver is gebaseerd op de python module bottle_.
M.b.v. dit raamwerk zijn eenvoudige webserver te bouwen. De webpagina bestaat uit HTML met bootstrap_ voor de
opmaak. Als thema is gekozen voor YETI_ van bootswatch_. Voor de applicatie logica wordt Angularjs_ gebruikt.

Op het moment van schrijven is alleen nog de basis lay-out van de applicatie plus een eerste opzet gereed.
De komende weken zal ik telkens een stukje functionaliteit toevoegen.

De eerste opzet is te vinden op mijn github repositorie: OnePageRESTApplication_
